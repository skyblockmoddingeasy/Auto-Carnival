# AutoCarnial
_A QOL module for hypixel skyblock._ 

## Features 
- Auto mode for each carnival game
- Full auto mode
- Auto grind masks
- Legit helpers
- All features are spec and wd safe

## Installation 
Download fabric api for 26.1.2 or 26.2

### Steps 
1. Download the lastest version of the mod
2. Move the jar into the `minecraft/mods` folder
3. Restart Minecraft

## Usage
_To open the GUI, the command is /carnival!_ --- Enjoy!
